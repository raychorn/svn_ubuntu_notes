ALPHACONTENT

Presentation | Author's notes | Install instructions | Upgrade instructions | SEF instructions | License | Changelog

Presentation :

AlphaContent is a component that creates a Yahoo style directory of content items (articles) in specified
sections/categories of your Joomla! website. AlphaContent also makes it possible to sort this listing
alphabetically based on the titles or intro of these articles. Many parameters for the presentation
and selection possible of the sections/catégories to be or not published. This component will be
appropriate more particularly for all those which use the articles like system of cards, or those which
need a classification of their items alphabetically or in the form of directory.

Author's notes :

From version 2.5, it is possible to create various menus attached to the component AlphaContent except
in the case where you use a component such as Sef Advance, OpenSEF, Artio JoomSEF... The problem comes owing to
the fact that these components they even seek to it Itemid of the menu and thus take into account only the
first menu attached to the found component. The menus can be parameterized in order to accept only certain
sections or specific categories what allows for example to create many distinct directories.
To parameterize correctly a menu, you must initially create new finely attached to the AlphaContent component
then to record this one. Publish then the menu and type the parameters.
These new parameters have priority with those published in the general configuration of the component.

Installation Instructions :

Installation of AlphaContent is easy, and is performed just like any other standard Joomla component installation.

In Joomla's administrative interface, go to Components -> Install/Uninstall.
On the bottom, you will see something called "Upload new component"
Click "Browse", then locate the folder to which you've downloaded AlphaContent and select the file com_alphacontent.zip.
Click the button "Upload File & Install" to confirm upload and installation of the AlphaContent package file.
Follow the instructions on the screen.
Go to menu item Components -> AlphaContent, and set the configuration.

Upgrade Instructions :

For upgrading from one version of AlphaContent to another version of AlphaContent, the only thing you need to do before uninstall
the old version is save the existing configuration file (named "alphacontent_config.php" in admininistrator>components>com_alphacontent) and install the new version.
After installing the new version, replace settings introduced in the new version by your "alphacontent_config.php" file.
Go to menu item Components -> AlphaContent, and set the configuration for apply new parameters.

SEF instructions :

Artio JoomSEF : custom name (AlphaContent menu) = alphacontent

Sef Advance : custom name (AlphaContent menu) = same custom name in settings Sef Advance component

OpenSEF : custom name (AlphaContent menu) = same custom name in settings OpenSEF component
If error : edit the administrator\components\com_sef\core\sef.resolver.php line 390
replace
$this->output .= ($url[0] != '&' ? '&' . $url : $url);
by
@$this->output .= ($url[0] != '&' ? '&' . $url : $url);
